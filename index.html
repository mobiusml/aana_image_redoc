Aana API (0.1.0)

Download OpenAPI specification:

Aana API for image search and retrieval in user-defined collections. This API allows users to index images, search for images using natural language queries, and recommend images based on text content.

First, you need to create a collection. Then use the Index endpoint, which generates rich metadata and stores it in a vector database. This enables powerful Search functionality, allowing you to retrieve relevant content from your collection using natural language queries.

Collections Management

Images are grouped into collections, which must be created before indexing or searching can take place. This set of endpoints allows you to create, update, and delete collections.

Webhooks

Aana provides a webhook system that allows you to receive real-time notifications for key task events. This is particularly useful for integrating Aana with your existing workflows.

Webhooks are HTTP callbacks that are triggered by specific events in the Aana system. When an event occurs, Aana sends a POST request to the specified webhook URL with details about the event. Currently, Aana supports the following events:

  • task.completed: Triggered when a task is completed successfully.
  • task.failed: Triggered when a task fails.
  • task.started: Triggered when a task starts processing.

openai-api

Chat Completions

Handle chat completions requests for OpenAI compatible API.

Authorizations:
ApiKeyAuth
Request Body schema: application/json
required
model
required
string (Model)

The model name (name of the LLM deployment).

required
Array of objects (Messages)

A list of messages comprising the conversation so far.

temperature
number (Temperature) >= 0
Default: 1

Float that controls the randomness of the sampling. Lower values make the model more deterministic, while higher values make the model more random. Zero means greedy sampling.

top_p
number (Top P) ( 0 .. 1 ]
Default: 1

Float that controls the cumulative probability of the top tokens to consider. Must be in (0, 1]. Set to 1 to consider all tokens.

max_tokens
integer or null (Max Tokens) >= 1

The maximum number of tokens to generate.

repetition_penalty
number (Repetition Penalty)
Default: 1

Float that penalizes new tokens based on whether they appear in the prompt and the generated text so far. Values > 1 encourage the model to use new tokens, while values < 1 encourage the model to repeat tokens. Default is 1.0 (no penalty).

stream
boolean or null (Stream)
Default: false

If set, partial message deltas will be sent, like in ChatGPT. Tokens will be sent as data-only server-sent events as they become available, with the stream terminated by a data: [DONE] message.

Responses

https://v1.api.aana.ai/chat/completions
